Detailed Description

Interface for display observer implementations.

The IDisplay class defines the contract for any display that observes a vehicle. When the vehicle's state changes, the Vehicle class calls Update() on all attached observers. Classes implementing this interface must provide the Update() behavior appropriate for their display type.
