Detailed Description

Abstract base class for all meter display components.

The Meter class provides the foundation for visual display elements that react to state changes from a subject (for example, a Vehicle). It maintains a WindowsDisplay instance used for rendering.

Display pointers are stored using move semantics. Null pointers are not permitted and will result in an exception. Subclasses should rely on the protected member m_window when performing drawing operations.
